# Absence Rate Calculator

Calculate student absence rates for a single student or an entire class using a Flask backend and React frontend. This project is designed for teachers or school administrators to quickly determine absence percentages efficiently.

---

## Description

This application allows users to calculate student absence rates. Users can choose to calculate for a single student or an entire class. The backend handles all the calculation logic, while the frontend provides a user-friendly interface. The absence rate is displayed as a percentage along with possible student days and actual days that students had come.

---

## Features

- Calculate absence rate for **one student** or **whole class**
- Dynamic input forms based on selected mode
- Real-time calculation and display of results
- Backend API built with **Flask**
- Frontend built with **React**

## Installations

### Backend
1. Go to the backend folder
```bash
cd backend
```

2. Create Virtual Enviroment
```bash
python -m venv venv
```

3. Activate Virtual Enviroment

#### Windows:

```bash
venv/Scripts/activate
```

#### Mac/Linux:

```bash
source venv/bin/activate
```
4. Install dependencies

```bash
pip install -r requirments.txt
```

5. Run backend

```bash
python3 app.py
```

### Frontend (Another terminal)
1. Go to frontend folder
```bash
cd frontend
```
2. Install dependencies
```bash
npm install
```

3. Run frontend
```bash
npm start
```
